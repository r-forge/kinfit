Dataset D from the FOCUS Kinetics report from 2006

Usage

data(FOCUS_2006_D)

Dataset D from the FOCUS Kinetics report from 2006

Description

Data taken from an “existing dataset”. At each time point two data points were generated, as the experiment was performed in duplicate.

Format

A data frame with 10 observations on the following 2 variables.

t
a numeric vector containing time points

parent
a numeric vector containing parent concentrations in percent of applied radioactivity

m1
a numeric vector containing concentrations of metabolite 1 in percent of applied radioactivity
